# Claude.ai — Theme
---

## Palette

| Role | Color (HEX) | Use / Notes |
|---|---:|---|
| Background (main) | `#1C1917` | Dark warm black — primary app background |
| Sidebar / Panels | `#292524` | Warm dark brown — elevated panels, sidebars |
| Elevated Surface | `#3D3936` | Surfaces above background (cards, modals) |
| Text (primary) | `#F5F0EB` | Off-white — primary body text |
| Text (secondary) | `#A8A29E` | Warm gray — secondary text and labels |
| Placeholder / Muted | `#78716C` | Muted gray — placeholders and disabled text |
| Accent / Brand | `#D97757` | Claude Orange — primary brand / CTA |
| Accent Hover | `#C96644` | Orange hover state |
| Highlight / Sand | `#E8C9A0` | Warm tan — highlights, badges, subtle accents |
| Border / Divider | `#44403C` | Thin dividers and subtle borders |
| Input Fields | `#57534E` | Input backgrounds, inactive controls |
| Subtle overlay (5%) | `#F5F0EB0D` | 5% off-white overlay for glass/overlay effects |

## Accessibility & contrast

- Primary text (`#F5F0EB`) on main background (`#1C1917`) provides strong contrast for readability in dark UI.
- Use `--text-secondary` and `--text-muted` only for less prominent content (labels, helper text). Verify contrast for small text (< 18px) approaches WCAG AA (4.5:1). If needed, increase size or weight.
- Accent (`--brand`) should be used for CTAs and icons. Ensure `--brand` over `--bg-panel` meets clickable element contrast; if failing, add a subtle outline or brighter highlight (`--highlight`) behind text.

---

## Figma / Design-tool usage

- Name color tokens exactly as the variables above (`bg/main`, `text/primary`, `brand/default`, `brand/hover`, etc.) to keep a consistent mapping between code and design files.
- Provide both hex and 8-digit hex (for the overlay) to designers: `#F5F0EB` and `#F5F0EB0D`.

---

## Quick guidelines

- Use `--bg-main` for page backgrounds; `--bg-panel` for cards and panels; `--surface-elevated` for elevated components (modals, dialogs).
- Reserve `--brand` / `--brand-hover` for primary CTAs and brand emphasis.
- Use `--highlight` sparingly to call attention (badges, selected states).
- `--overlay-5` is useful for glass/blur overlays or subtle disabled-state overlays.
